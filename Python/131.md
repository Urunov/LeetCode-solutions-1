# 131. Palindrome Partitioning

## Solution 1

```python
# DFS
class Solution(object):
    def partition(self, s):
        """
        :type s: str
        :rtype: List[List[str]]
        """
        self.ans = []
        self.dfs(s, "", [])
        return self.ans

    def dfs(self, s, cur_str, cur_ans):
        if s == "":
            if len(cur_str) != 0:
                return
            self.ans.append(cur_ans[:])
            return
        cur_str = cur_str + s[0]
        if cur_str == cur_str[::-1]:
            self.dfs(s[1:], "", cur_ans + [cur_str])
        self.dfs(s[1:], cur_str, cur_ans)
```
