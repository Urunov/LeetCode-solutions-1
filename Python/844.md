# 844. Backspace String Compare

## Solution 1 (O(m+n))

```python
class Solution(object):
    def backspaceCompare(self, S, T):
        """
        :type S: str
        :type T: str
        :rtype: bool
        """
        def build(s):
            ans = []
            for ch in s:
                if ch != "#":
                    ans.append(ch)
                elif ans:
                    ans.pop()
            return "".join(ans)

        return build(S) == build(T)
```
