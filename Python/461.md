# 461. Hamming Distance

## Solution 1

```python
class Solution(object):
    def hammingDistance(self, x, y):
        """
        :type x: int
        :type y: int
        :rtype: int
        """
        xor = x ^ y
        ans = 0
        while xor:
            if xor & 1:
                ans += 1
            xor = xor >> 1
        return ans
```
