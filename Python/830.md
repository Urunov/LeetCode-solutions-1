# 830. Positions of Large Groups

## Solution 1 (time O(n), space O(1))

```python
class Solution(object):
    def largeGroupPositions(self, s):
        """
        :type s: str
        :rtype: List[List[int]]
        """
        ans = []
        prev_ind = 0
        for i in range(1, len(s)):
            if s[i] != s[i - 1]:
                if i - prev_ind >= 3:
                    ans.append([prev_ind, i - 1])
                prev_ind = i
        if len(s) - prev_ind >= 3:
            ans.append([prev_ind, len(s) - 1])
        return ans

```
