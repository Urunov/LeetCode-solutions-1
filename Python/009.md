# 9. Palindrome Number

## Solution 1

```python
class Solution(object):
    def isPalindrome(self, x):
        """
        :type x: int
        :rtype: bool
        """
        if x < 0:
            return False
        x1, x2 = 0, x
        while x > 0:
            x1 = x1 * 10 + x % 10
            x //= 10
        return x1 == x2
```

## Solution 2

```python
class Solution(object):
    def isPalindrome(self, x):
        """
        :type x: int
        :rtype: bool
        """
        s = str(x)
        return s == s[::-1]
```
